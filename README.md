# Favgalgirls

Static site for the Galgame heroine matching quiz.

## GitHub Pages

Use these settings in GitHub:

1. Open `Settings -> Pages`
2. Under `Build and deployment`, choose `Deploy from a branch`
3. Branch: `main`
4. Folder: `/(root)`

After GitHub finishes the deployment, the site URL will be:

- `https://krkree.github.io/Favgalgirls/`

The Chinese-name editor page will be:

- `https://krkree.github.io/Favgalgirls/name-editor.html`

## Local Notes

- `index.html` is the main quiz page
- `name-editor.html` is the manual Chinese-name editor
- `data/site-data.js` contains the generated quiz dataset
- `assets/characters/` contains heroine images
